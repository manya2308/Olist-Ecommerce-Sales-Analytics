# 🛒 Olist E-Commerce Sales Analytics

## 📌 Project Overview

This project analyzes Brazilian e-commerce sales data from Olist to understand sales performance, customer behavior, and business trends.

The objective of this project is to transform raw transactional data into meaningful business insights using SQL and Power BI.

The analysis focuses on identifying revenue drivers, customer purchasing patterns, and key performance indicators that can help businesses make data-driven decisions.

---

# 🎯 Business Objectives

The project aims to answer:

- Which product categories generate the highest revenue?
- How does sales performance change over time?
- Which states contribute the most to overall sales?
- What are the characteristics of customers and their purchasing behavior?
- How do customer reviews vary across different categories?

---

# 🛠️ Tools & Technologies Used

| Tool | Purpose |
|---|---|
| SQL Server | Data extraction, transformation, and business analysis |
| Power BI | Interactive dashboards and data visualization |
| Excel / Power Query | Data cleaning and preprocessing |

---

# 📂 Dataset

Dataset used:

**Brazilian E-Commerce Public Dataset by Olist**

The dataset contains information about:

- Orders
- Customers
- Products
- Sellers
- Payments
- Reviews
- Order Items

The data represents e-commerce transactions between 2016 and 2018.

---

# 🔄 Data Preparation Process

The workflow followed:

1. Imported raw CSV datasets into SQL Server.
2. Cleaned and transformed data for analysis.
3. Created relationships between tables.
4. Developed SQL queries to answer business questions.
5. Connected analyzed data with Power BI.
6. Built interactive dashboards for decision-making.

# 🔍 Key Insights

Some important findings from the analysis:

- A small number of product categories contribute a significant portion of revenue.
